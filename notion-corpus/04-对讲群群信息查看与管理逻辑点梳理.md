# 04 对讲群「群信息查看与管理」（US-群-02）逻辑点梳理

**核心命题：入群的是设备，操作的是设备背后的归属账号。**

## 一、核心模型与角色
每台终端设备（卡号/addr）是一个群成员；界面里做操作的是**设备背后的归属账号**。群里只有两个角色：
- **群主**：创建该对讲群的账号，拥有**最高权限**。
- **群成员归属账号**：在群内拥有 ≥1 台成员设备的账号，只能管理**自己名下的设备**。

群成员归属账号可以就是群主本人（按群主最高权限处理）。
---
## 二、角色与权限
权限为**按钮级控制**：无权操作时**直接隐藏入口**，不显示禁用态、不报错；Web 端与小程序权限完全对齐。
| 操作 | 群主 | 群成员归属账号（非群主时） |
|---|---|---|
| 查看群名称 | ✅ | ✅ |
| 编辑群名称 | ✅ | ❌ |
| 查看成员列表 | ✅ | ✅ |
| 编辑成员备注（群内昵称） | ✅ 所有成员设备 | ✅ 仅自己名下设备 |
| 移除成员 | ✅ 任意成员设备 | ✅ 仅自己名下设备 |
| 添加成员（邀请） | ✅ | ❌ |
| 查看邀请记录 | ✅ | ❌ |
| 结束群组 | ✅ | ❌ |
---
## 三、入口与主流程
- **入口 A（顶部套餐余额）**：默认落「套餐概览」Tab
- **入口 B（底部群信息）**：直达「群信息」Tab
---
## 五、设备群内备注（昵称）逻辑
| 规则 | 说明 |
|---|---|
| 入群初始化 | 设备首次加入该群时取快照：有备注→「备注(卡号)」；无备注→「卡号」 |
| 群级独立编辑 | 编辑仅对当前群生效，不回写设备绑定层备注 |
| 绑定层变更隔离 | 未退群期间，绑定层改备注不同步到群内昵称 |
| 跨群循环 | 退群再进其他群，重新取值，按群隔离 |
| 并发覆盖 | 最后修改为准 |
| 长度与显示 | 最长 20 字符，超出换行不截断 |
---
## 六、成员列表展示
- 列表项 = 群内每台终端设备；字段：头像（终端图标 + 状态）/ 群内昵称 / 设备卡号。
- 账号冻结/过期的成员设备**仍保留在列表**（不自动退群）。

### 6.1 头像 = 终端图标（实时同步，区别于备注）
头像是**实时读取设备当前图标**（与备注快照相反）；图标范围为**固定枚举**，不可自定义上传。

### 6.2 头像叠加：在线 / 报警 / 离线状态（全局统一）
状态为**设备全局状态**：与求救群共用同一套设备报警状态。
| 组合状态 | 主体图标 | 右上角角标 |
|---|---|---|
| 报警中 + 在线 | 红色（报警优先） | 绿色 ✓ |
| 报警中 + 离线 | 红色 | 灰色 × |
| 正常/已解除 + 在线 | 绿色 | 不展示 |
| 正常/已解除 + 离线 | 灰色 | 不展示 |
---
## 七、移除 / 离群操作（群信息内）
### 7.1 核心判定规则（这把尺）
- **账号「在不在」对讲群 = 该账号名下是否还有设备在本群。**
- 某设备离群后，看其归属账号名下是否还有设备在本群：
	- **未归零** → 账号仍在群、群继续。
	- **归零 + 非群主账号** → 账号退出本群，会话项灰「已退出」，**群继续**。
	- **归零 + 群主账号** → **仍是群主、群继续、会话项保持正常、可重新邀请**；**不再自动结束群**。
- **双重身份归属优先级**：离群/归零设备的归属账号同时就是群主本人时，一律按群主权限、走「归零 + 群主账号」分支。
- **群结束仅有两种情形**：① 群主在群信息内点**「结束群组」**；② **群主账号本身注销/删除**。设备归零本身不再触发结束群。

### 7.2 操作权限与确认
- 均需**二次确认**（文案含昵称/卡号）；**移除/退出不退还邀请星豆**；被移除/退出方不推送。
---
## 八、群已结束状态下的群信息
- **只读冻结口径（已确认）**：群结束后页面整体只读，**设备头像/在线·报警角标、设备套餐余额仍全局实时刷新**；**其余历史参数（群名、成员列表、群内昵称快照、消息记录等）一律定格为结束时刻快照，不再变化。**

### 8.1 会话项状态与可见性
- 群主只有「已结束 / 未结束」两态，**永远不会出现「已退出」**；只有非群主的普通成员归属账号才会出现「已退出」。

### 8.2 群结束的两条路径
- **路径①｜显式结束**：群主点「结束群组」→ 直接结束（US-群-03）。
- **路径②｜群主主体消失**：群主账号本身注销/删除 → 必然结束群。
- 单纯的设备移除/解绑/类型变更/换群即使造成群主设备归零，也不结束群。
---
## 九、关键规则速记
1. 入群的是设备，操作的是设备归属账号。
2. 群主=最高权限管全群；群成员归属账号=只能管自己设备，且可以就是群主本人。
3. 备注初始化取值一次，群级隔离、不回写、跨群重置、最后修改为准。
4. 头像=终端图标，实时同步；在线/报警/离线状态全局统一。
5. 权限按钮级控制：无权即隐藏，不禁用、不报错；Web 与小程序对齐。
6. 移除/退出的是单台设备；归零时：非群主=退出（灰「已退出」、群继续）；群主=群继续、仍是群主、可重新邀请（**设备归零不再自动结束群**）。
