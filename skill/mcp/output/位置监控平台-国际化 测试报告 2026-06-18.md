# 位置监控平台-国际化 测试报告 2026-06-18

> 报告生成时间：2026-06-18 18:00

## 一、测试结果

本轮测试已完成位置监控平台国际化项目的功能验证，当前仍有 **5 个未解决缺陷**，主要集中在 **【功能异常】 / 【国际化适配】** 等方向，其中 **2 个高优问题（二级）需优先修复**，3 个已延期问题建议下个版本跟进。

- **【功能异常】**：2 个二级问题直接影响核心业务流程。仅分配设备后二级账号设备列表查询为空，导致二级用户无法正常查看已分配设备；平台取消待发送消息后小程序仍显示"等待发送给终端"，存在消息状态不同步问题，影响用户操作判断
- **【国际化适配】**：3 个已延期四级问题涉及UI细节，包括状态角标过小导致在线状态难以识别、指令下发弹窗未过滤空分组、设备信息导出模版字体不适配国际版本
