# 【天翼】星联应急叫应平台2期 测试报告 2026-07-22

> 报告生成时间：2026-07-22

### 一、测试结果

本轮测试已完成 7 个模块的功能验证，当前仍有 **9 个未解决缺陷**（其中回归不通过 **2** 个），主要集中在 **资金/计费 / 数据一致性 / 消息/通信** 等方向，其中 **1 个高优问题（二级及以上）需优先修复**，**7 个已延期问题建议下个版本跟进**。

- **【资金/计费】：5 个未关闭**（含二级高优 #3493、#3370） 涉及设备套餐报位额度已用完且星豆余额为0时，SOS报警未被封禁仍可正常触发、纯个人账号无星豆无套餐时终端上下行未封禁，套餐进入欠费累加状态、套餐信息报位误显为次/小时、星豆余额与套餐报位剩余均为0时，终端上报的SOS位置报位仍正常入群未被拦截 等；可能造成资金或计费数据错误，影响对账与用户权益（影响待复核）。
- **【数据一致性】：2 个未关闭**（含二级高优 #3711） 涉及群主结束群聊后，被邀请成员侧状态误显示“已退出”且无法查看历史消息、报平安信息与报警信息列表中“位置”按钮字体样式不一致；数据展示或同步不一致，可能误导业务判断（影响待复核）。
- **【消息/通信】：7 个未关闭** 涉及设备进群后修改设备类型，设备未能自动退群、群聊结束后，群主在“对讲群邀请清成员操作”页面无法查看历史邀请记录、报警信息列表中未处理报警的“报警类型”和“报警内容”未使用红色高亮、数据不足一屏仍显示没有更多了 等；消息收发链路异常，可能导致漏收/重复收（影响待复核）。
- **【权限/安全】：2 个未关闭** 涉及账号与卡号一致时，终端上报报警消息显示在聊天室右侧、个人账号列表“锁定状态”字段为冗余字段，应移除；权限或边界校验缺失，存在安全/越界风险（影响待复核）。
- **【回归验证】**：另有 **18 个** 已修复缺陷待回归验证。
